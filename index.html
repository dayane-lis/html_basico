História dos computadores

(Voltar)

Computadores de Primeira Geração: Os computadores de primeira geração eram enormes, chegando a ocupar salas inteiras, e consumiam grande quantidade de energia, além de superaquecerem.

Seu funcionamento era a partir de vávulas eletrônicas e resolviam apenas uma tarefa por vez.

Temos como exemplos o Harvard Mark I, o Colossus e o ENIAC (Eletronic Numerical Integrator and Computer).

Este último, possuia 19.000 vávulas e consumia cerca de 200 quilowatts.


Computadores de Segunda Geração: Foi a geração de computadores onde se substituiu o funcionamento com válvulas por transistores, o que ajudou a agilizar seu processamento.

Foram os primeiros a armazenar instruções em suas memórias.

Um computador que representa a Segunda geração é o IBM 1401, que era usado como periférico.


Computadores de Terceira Geração: Esta é primeira geração onde o usuário utilizava teclados e monitores na interação com o computador (computadores de mesa).

Um dos maiores avanços foi a criação dos circuitos integrados, o que ajudou a melhorar a velocidade e a eficiência das máquinas.

A partir destes avanços, surgiram coputadores capazes de executar mais de uma tarefa ao mesmo tempo.


Computadores de Quarta Geração: A partir daqui os computadores se tornam mais compactos e com um aumento da capacidade de processamento e velocidade.

Tivemos como avanços neste período a interface gráfica do usuário, a criação do mouse, bem como o desenvolvimento e evolução de dispositivos portáteis.

Já se fala que nos encontramos na Quinta Geração, devido ao desenvolvimento da Inteligência Artificial.


Tags HTML

(Voltar)

As tags são utilizadas para produzir o conteúdo visual de uma página web, pois seus respectivos comandos são interpretados pelo navegador, de modo a formatar tamanho de fonte, posição de textos e figuras, inclusão de listas, acesso a links tanto internos quanto externos etc.

Podemos dizer, então, que as tags são responsáveis por toda a parte visual de uma página.

A própria estrutura do nosso arquivo HTML é formado por tags para delimitar cabeçalho, corpo e rodapé, além das tags de título (h1,h2,h3..), de parágrafo (p), pular linha (br), dentre outras.


Atributos básicos

(Voltar)

As propriedades das tags são chamadas de Atributos. Existem atributos mais genéricos, como por exemplo, atributo de classe (class) ou de estilo (style) através do qual podemos utilizar formatação CSS dentro do nosso HTML.

Também temos atributos mais específicos, como o atributo type utilizado dentro da tag input para especificar que apenas números poderão ser escritos dentro do input.

Outro exemplo de atributo específico é o scr, utilizado para incluir o link para uma figura dentro da tag img, assim como o atributo width que determina um tamanho para a figura.


Listas ordenadas e não ordenadas

(Voltar)

Listas ordenadas ordenam e numeram os itens incluídos nela de maneira automática, independente de associarmos números aos mesmos.

Listas não ordenadas utilizam marcadores para listar os itens, por padrão é utilizado o marcador de bolinhas. Porém, é possível alterar esse padrão através do CSS, bem como podemos retirar esses marcadores.

Listas não ordenadas podem ser utilizadas quando precisamos fazer um menu horizontal no site, graças a possibilidade de alteração através do CSS.
